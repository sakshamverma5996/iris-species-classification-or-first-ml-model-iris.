# Iris Flower Classification Model 🌸

This is my very first Machine Learning project! I built a classification model to identify flower species based on sepal and petal measurements.

## 📈 Performance
* **Total Test Samples:** 34
* **Correct Predictions:** 33
* **Accuracy Rate:** 97.05% 🚀

## 🛠️ Tech Stack
* Python
* Scikit-Learn
* Pandas / NumPy
* Seaborn (for data visualization)

## 🧠 What I Learned
Working with this classic dataset helped me understand how classification algorithms draw decision boundaries, especially when dealing with the feature overlap between *Iris versicolor* and *Iris virginica*.
